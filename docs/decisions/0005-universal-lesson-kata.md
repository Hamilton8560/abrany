## Context
Lessons must fit every domain — skills (coding, language, math), knowledge (geopolitics,
books), interpretive subjects (Bible, philosophy), memorization, and exam prep.

## Decision
Every milestone's lessons follow one fixed arc — Read → Teach → Practice → Apply → Check →
Review — whose stage *meaning* adapts to the learning mode (e.g. "practice" = drills for a
skill but reflection for an interpretive subject; "check" = quiz vs. self-reflection vs. mock
exam).

## Consequences
One backbone for all subjects; verified it reshapes correctly for a skill (Spanish) and an
interpretive subject (48 Laws of Power). A rigid identical template is avoided by mode-adaptation.
